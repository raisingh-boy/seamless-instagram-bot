## Как писать контент

1. **Одна статья = один источник.** Не мешать несколько статей в один пост.
2. **Три формата с одного источника:**
   - Пост для Instagram (@seamless_research)
   - Пост для TikTok (адаптированный)
   - Пост для YouTube (расширенный)
3. **Язык — только английский.** Никакого русского.
4. **Призывы — generic.** Не использовать @упоминания каналов, если не уверен. Вместо этого: "Subscribe", "Follow for more", "Drop a comment".
5. **Структура YouTube:** содержательно, не коротко. Развёрнутое объяснение концепции, не список.
6. **TikTok:** коротко, ёмко, с призывом подписаться/сохранить. Без таймингов и режиссуры.

## Учёт статей

1. Записывать все найденные статьи в articles/TRACKING.md
2. Отмечать какие использованы (USED), какие нет (NOT USED)
3. После создания контента — обновлять статус
4. Систематично, последовательно

## Как скачивать с Instagram

- **Метод:** Instagram GraphQL API
  - Playwright → получаем csrftoken + ig-app-id
  - POST /graphql/query с doc_id=24368985919464652
  - JSON: data > xdt_api__v1__media__shortcode__web_info > items[0] > video_versions > url
- **Скрипт:** `/root/.openclaw-instagram/download_ig_post.py`
- Видео скачивается с ftyp-заголовком, конвертируется под TikTok
